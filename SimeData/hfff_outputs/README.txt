This folder contains HFFF-compatible data input files for the **entire** referenceIntObservationsFull.csv dataset, each processed using one of the three methodologies described in the deep research report:

1. 1.hfff_binary.csv — Binary Heuristic Thresholding (from_binary_confidence)
2. 2.hfff_cwci.csv — Continuous Weighted Confidence Index (from_confidence)
3. 3.hfff_discount.csv — Probabilistic Discounting via Uniform Redistribution (from_counts)
4. 4.hfff_composite.csv — Composite file with all three vectors side-by-side for each point

See HFFF_Methodology_Report.md in this folder for a summary, recommendations, and methodology details.
All files are HFFF-compatible and ready for downstream use.
